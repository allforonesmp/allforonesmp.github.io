AutoModpack

Backend

Automatically sends every required client-side mod (minimap, chat heads, etc.) to your game the moment you connect, so you never have to hunt down or install mods yourself.

automodpack.jar
How to use▾
  • Just join with a vanilla Fabric install for Minecraft 26.2 — no mods needed beforehand.
  • On first connect you'll get a prompt to download the modpack. Accept it and restart when asked.
  • Whenever the server's mods update, you'll be prompted again automatically.

Fabric API

Backend

The foundational library that almost every other mod on the server depends on. It doesn't add any features on its own.

fabric-api-0.154.2+26.2.jar
How to use▾
  • Nothing to do — it just needs to be present for other mods to run.
  • AutoModpack installs the matching client copy for you automatically.

Fabric Language Kotlin

Backend

Adds Kotlin language support to Fabric so mods written in Kotlin (like EnderNexus) can run on the server.

fabric-language-kotlin-1.13.12+kotlin.2.4.0.jar
How to use▾
  • Purely a background dependency — no player interaction at all.

Terralith

Terralith_26.2_v2.6.4.jar

Fast IP Ping

fast-ip-ping-v1.0.11

EasyAuth

Player

Handles login and registration since the server runs in offline mode. Protects your username from being impersonated by anyone else.

easyauth-mc26.2-3.4.3-SNAPSHOT.49.jar
How to use▾
  • First time joining: /register <password> <password>
  • Every time after that: /login <password>
  • You have a limited number of login tries and a short window to log in before being kicked — have your password ready.

SkinRestorer

Player

Restores your real Minecraft skin even though the server runs offline mode, so everyone still looks like themselves.

skinrestorer-2.9.0+26.1-fabric.jar
How to use▾
  • /skin set mojang/ely.by/web <username/link> — pull in a specific skin.
  • /skin clear — reset back to default.
  • Your own skin is applied automatically on join in most cases.

Open Parties and Claims

Player

Lets you protect your base by claiming land, and form parties with friends so they can build and interact inside your claim.

open-parties-and-claims-fabric-26.2-0.27.7.jar
How to use▾
  • Open the claims map with the default keybind (configurable in Controls), or use /claim.
  • Create a party: /openpac-parties create.
  • Invite a friend: /openpac-parties member invite <player>.
  • By inviting members to your party, it allows them to break, place and interact with any blocks/entites in your claim and vice versa for them.
  • Delete Party: /openpac-parties destroy confirm.

Dungeons and Taverns

dungeons-and-taverns-5.3.0.jar
How to use▾
  • You don't need to do anything it will generate for you!

Veinminer

Player

Mine an entire connected ore vein in one go instead of block by block.

Veinminer-fabric-3.1.3+mc26.2.jar
How to use▾
  • Hold the activation key (default is bound to sneak/toggle in your Controls menu, under "Veinminer") while breaking an ore block.
  • The whole connected vein breaks at once, as long as you have enough durability/inventory space.
  • Works only on configured "vein-mineable" blocks like ores, not general stone.

MobFilter

Backend

Controls which mobs are allowed to spawn on the server, tuned by the admins for a smoother, less chaotic world.

mobfilter-fabric-0.28.0+26.2.jar
How to use▾
  • Runs entirely server-side based on a config file — nothing for players to configure.
  • You may simply notice certain mobs spawn less or not at all.

EndLock

Backend

Controls when and how the End dimension can be accessed, keeping End-game progression paced for the whole server.

endlock-1.0.3.jar
How to use▾
  • No commands for players — if the End portal is locked, you'll see a message explaining why when you try to use it.

Vanish

Staff Only

Lets staff go completely invisible to other players — useful for quietly checking in on the server without being seen.

vanish-1.6.15+26.2.jar
How to use▾
  • /vanish — toggle invisibility on/off.

ServerReplay

Staff Only

Records replays directly on the server so events, builds, or moments can be captured and watched back later — no client-side recording mod needed.

ServerReplay-3.5.0+26.2.jar
How to use▾
  • /replay start — begin recording.
  • /replay stop — end and save the recording.
  • Requires staff/operator permission; finished replays are saved on the server for later viewing.

Xaero's Minimap

Client

Shows a live minimap in the corner of your screen with nearby terrain, players, and waypoints.

xaerominimap-fabric-26.2-26.2.0.jar
How to use▾
  • Installed automatically by AutoModpack — appears in your corner once connected.
  • Press the minimap settings key (default in Controls under "Xaero's Minimap") to resize or reposition it.
  • Set a waypoint with /way or through the in-game waypoint menu.
